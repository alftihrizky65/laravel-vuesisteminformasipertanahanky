# Sistem Informasi Pertanahan

## Deskripsi
Sistem Informasi Pertanahan adalah aplikasi berbasis web yang digunakan untuk mengelola data pertanahan secara digital. Sistem ini membantu dalam pengolahan data tanah, pemilik tanah, serta informasi sertifikat secara lebih terstruktur, cepat, dan efisien.

Aplikasi ini dibuat sebagai proyek pengembangan sistem informasi menggunakan framework modern untuk meningkatkan efisiensi pengelolaan data pertanahan.

---

## Fitur Utama
- Login dan autentikasi pengguna
- Dashboard sistem
- Manajemen data pemilik tanah
- Manajemen data tanah
- Manajemen sertifikat tanah
- Pengolahan dan penyimpanan data pertanahan
- Sistem database terstruktur

## Cara Menjalankan Project

1. Clone repository

```
git clone https://github.com/alftihrizky65/laravel-vuesisteminformasipertanahanky.git
```

2. Masuk ke folder project

```
cd laravel-vuesisteminformasipertanahanky
```

3. Install dependency backend

```
composer install
```

4. Install dependency frontend

```
npm install
```

5. Copy file environment

```
cp .env.example .env
```

6. Generate key aplikasi

```
php artisan key:generate
```

7. Jalankan server Laravel

```
php artisan serve
```

8. Jalankan Vue / Vite

```
npm run dev
```
